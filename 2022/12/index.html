Git Cheatsheet

December 16, 2022
tutorial
Introduction I record some useful tips and skills on working with git. Submodule How to prevent it switching to a detached HEAD on updating the submodule It can be fixed by setting the remote tracking branch with git submodule set-branch -b <branch name> <submodule path>. The <branch> name should not contain the remote repo alias. E.g., the command of git submodule set-branch -d main vim will make the submodule vim track the changes of remote main branch.
